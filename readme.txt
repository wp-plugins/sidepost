To add recent posts in sidebar, with thumbnails, from specific categories...

== Description ==

There are many plugins for adding recent posts in sidebar, I know... But this is my version. Its easy to use and has all the right settings. 

For instance, you can choose posts from specific categories, number of posts, thumbnail size, number of words in excerpt...

== Installation ==

1. Upload the folder `srp` to the directory `/wp-content/plugins/`.
2. Activate the plugin via the 'Plugins' menu in your WordPress admin panel.
3. You can adjust the necessary settings in your WordPress widget area.

== Frequently Asked Questions ==

= How to get posts from more than one categorie? =

To get posts from more than one categorie, just write more than one ID number and use commas to separate.
